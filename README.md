## Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

## Development

Run the development server:
```bash
npm run dev
```
Then open [http://localhost:3000](http://localhost:3000)

## Production

Build and start the production server:
```bash
npm run build
npm run start
```
